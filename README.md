# robustAR
This R package provides the implementation of the robust AR statistic introduced in Klooster and Zhelonkin (202X).

## Installation
To install `robustAR`, use the function `install_github()` of the `devtools` package.

```R
install.packages("devtools")
devtools::install_github("Jens-Klooster/robustAR")
library(robustAR)
```

After the installation, the robustAR vignette provides further examples of how to use the function `robustAR`.
